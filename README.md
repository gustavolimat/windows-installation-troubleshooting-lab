# Windows Installation & Basic Troubleshooting Lab

## Objective
Perform a clean installation of Windows OS and configure the system for basic end-user usage, following best practices commonly used in IT Support environments.

## Environment
- Operating System: Windows 11 (Virtual Machine)
- Installation method: ISO / Virtual Machine
- Installation type: Clean installation
- Platform: Virtual environment using VirtualBox

## Project Structure / Screenshots
### 01_vm_setup
This folder contains screenshots of the Virtual Machine setup, including:
- Creation of the VM
- Allocation of RAM, CPU, and disk space
- Preparation of the VM for Windows installation

### 02_windows_install
Screenshots of the Windows installation process:
- Language and region selection
- Disk partitioning
- Installation progress
- First login and desktop setup

### 03_configuration
Screenshots of post-installation configuration:
- User account creation
- System information (About)
- Device Manager
- Windows Update

### 04_network
Screenshots demonstrating network setup and testing:
- IP configuration
- Network connectivity tests (ping)

### 05_troubleshooting
Screenshots showing troubleshooting steps:
- Network adapter not detected
- Driver installation and updates
- Confirming issues are resolved

## Steps Performed
- Created a bootable ISO / VM for Windows installation
- Configured VM settings (RAM, CPU, disk)
- Performed a clean installation of Windows
- Created local user accounts (standard and administrator)
- Installed system drivers
- Applied Windows updates
- Configured basic security settings (Windows Defender and Firewall)

## Troubleshooting
- **Issue:** Network adapter not detected after installation  
  **Solution:** Installed the correct driver from the manufacturer website
- **Issue:** Missing or outdated drivers  
  **Solution:** Used Device Manager to identify and update drivers

## Skills Demonstrated
- Windows OS installation
- Basic system configuration
- Troubleshooting hardware and software issues
- User account management
- IT documentation

## Notes
This project was completed as a hands-on lab to reinforce practical IT Support skills, focusing on real-world scenarios commonly encountered in entry-level support roles.
